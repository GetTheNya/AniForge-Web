**AniForge Web** — браузерний компаньйон [додатку AniForge для Android](https://github.com/GetTheNya/AniForge). Той самий офлайн-каталог аніме, тепер прямо у твоєму браузері. Жива версія доступна за посиланням [aniforge.pages.dev](https://aniforge.pages.dev).

Жодних встановлень. Жодних розширень. Просто відкрий сторінку і переглядай понад 22 000 записів аніме миттєво.

## Що можна робити?

### 🔍 Пошук та фільтрація каталогу
Досліджуй кожен запис аніме з потужною фільтрацією — за жанром, тегом, студією, форматом, статусом трансляції, першоджерелом, діапазоном оцінок та кількістю серій. Сортуй за оцінкою, популярністю, роком, назвою або відповідністю.

### 📋 Відстежуй своє аніме
Увійди через Google, щоб керувати своїми списками — **Дивлюся**, **Переглянуто**, **У планах**, **Призупинено** та **Закинуто**. Оновлюй прогрес серій, ставлю оцінки та залишай особисті нотатки.

### 📁 Власні колекції
Створюй власні тематичні добірки аніме — групуй як тобі зручно та керуй ними зі сторінки Бібліотеки.

### 🖼️ Детальна інформація про аніме
Кожне аніме має повну сторінку з обкладинкою, описом, трейлерами, скриншотами, інформацією про серії, жанрами, тегами, студіями, персоналом, хронологією франшизи та пов'язаними роботами.

### 🌳 Слідкуй за франшизами та хронологією
Дивись приквели, сиквели, спін-офи та альтернативні версії на хронології франшизи — жоден пов'язаний проєкт не загубиться.

### 🌐 Двомовний інтерфейс
Повна локалізація англійською та українською мовами, з можливістю пріоритезації українських назв аніме з каталогу.

### 📱 Адаптовано для мобільних
Мобільні відвідувачі бачать сторінку завантаження нативного додатку для Android із QR-кодом та можливістю одним дотиком перейти до веб-версії.

## Як це працює

AniForge Web запускає справжню базу даних **SQLite прямо у браузері** через WebAssembly. При першому візиті каталог завантажується потоково та розпаковується у браузері — після цього застосунок стартує миттєво з Cache API. Фонова перевірка версій тримає каталог актуальним з атомарною заміною A/B слотів, як і в Android-додатку.

Дані відстеження (списки, оцінки, колекції) синхронізуються через **Supabase** з автентифікацією Google, а каталог працює повністю офлайн.

## Стек технологій

| Шар | Технологія |
|---|---|
| Фреймворк | React 19 + TypeScript |
| Збірник | Vite 8 |
| Стилі | Tailwind CSS 4 |
| База даних | sql.js (SQLite скомпільований у WASM) |
| Дані користувача | Dexie (IndexedDB) + Supabase |
| Автентифікація | Supabase Google OAuth |
| i18n | i18next + визначення мови браузера |
| Лінтинг | Oxlint |

## Початок роботи

```bash
# Клонуй репозиторій
git clone https://github.com/GetTheNya/AniForge-Web.git
cd AniForge-Web

# Встанови залежності
npm install

# Запусти dev-сервер
npm run dev
```

Застосунок буде доступний за адресою `http://localhost:5173`.

### Збірка для продакшну

```bash
npm run build
npm run preview
```

## Структура проєкту

```
src/
├── components/        # UI-компоненти (AnimeCard, FilterPanel, Layout тощо)
├── context/           # React-контексти (Auth, Database, Settings, Tracking тощо)
├── hooks/             # Кастомні хуки (useAnimeSearch, useAnimeDetail тощо)
├── services/          # SQLite WASM рушій, Supabase клієнт, білдер запитів
├── types/             # Визначення типів TypeScript
├── utils/             # Утиліти
├── i18n.ts            # Конфігурація інтернаціоналізації (EN + UK)
├── App.tsx            # Кореневий компонент
├── main.tsx           # Точка входу
└── index.css          # Глобальні стилі та дизайн-токени
```
